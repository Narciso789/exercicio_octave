Exercicio

%Considere um composito plano de fibra de vidro e matriz de resina epoxi, com
%fracao volumetrica de 45% de fibra, carregado como mostrado abaixo. A
%configuração do laminado é dada como sendo [+45/0/90]t e admite-se a espessura
%de cada camada como sendo de 0,2 mm. Considerando as propriedades dadas abaixo,
%determinar:

% a) as equacoes constitutivas do laminado;

% b) as deformações centrais e curvaturas da placa;

% c) as deformacoes em cada camada nas direcoes globais (X-Y) e direcoes
%principais (1-2);

% d) as tensoes em cada camada nas direcoeses globais (X-Y) e direcoes
%principais (1-2);

% e) a possivel ocorrencia de falha usando o criterio de tensao maxima e o
%criterio de Azzi-Tsai-Hill.

%Dados:

%E1 = 39 GPa

%E2 = 8,3 GPa

%G12 = Gpa

%v 12 = 0,26

%Resistencia a tracao na direcao 1 (Xt) = 1062 MPa

%Resistenciancia a compressao na direção 1 (Xc) = 958 MPa

%Resistencia compressao na direção 2 (Yc) = 118 MPa

%Resistencia ao cisalhamento no plano (S12) = 78 MPa

EXERCICIO RESOLVIDO

% Dados de entrada

E1 = 39;  %GPa
E2 = 8.3; %GPa
G12 = 4.1;  %GPa
v12 = 0.26;
Xt = 1.062; %GPa
Xc = 0.958;  %GPa
Yt = 0.031; %GPa
Yc = 0.118; %GPa
S12 = 0.078;  %GPa
z_k0 = -0.0003; %N/m
z_k1 = -0.0001; %N/m
z_k2 = 0.0001; %N/m
z_k3 = 0.0003; %N/m

% Carregamento
Nx = 200000;   %N/m
Ny = 100000;   %N/m
Nxy = 0;       %m/m

Calculo o coeficiente de Poisson Menor (v21):

v21 = v12*E2/E1;

D. As matrizes [Q] e [S] da lamina considerando as fibras alinhadas ao eixo longitudinal da lamina (teta = 0, x=1):

Calculo dos elementos da matriz rigidez de laminas ortotropicas - (GPa):

Q11 = (E1/(1-(v12*v21)));
Q12 = (v21*E1)/(1-(v21*v12));
Q21 = (v12*E2)/(1-(v12*v21));
Q22 = (E2)/(1-(v21*v12));
Q66 = G12;

% Calculo dos elementos da matriz de flexibilidade de laminas ortotropicas - (GPa)?1:
S11 = (1/E1);
S12 = (-v21)/(E2);
S21 = (-v12)/(E1);
S22 = (1)/(E2);
S66 = (1)/(G12);

Matriz de Rigidez de laminas Ortotropicas em EPT - (GPa):

Q = [Q11, Q12, 0; Q21, Q22, 0; 0, 0, Q66];

Matriz de Flexibilidade de laminas Ortotropicas em EPT - (GPa)?1:

LEMBRAR QUE A MATRIZ (S) PODE SER CALCULADA POR: "S = INV(Q)"

S = [S11, S12, 0; S21, S22, 0; 0, 0, S66];

E. As matrizes de Rigidez [Q_barra] e Flexibilidade [S_barra] considerando qua as fibras ao longo da direcao 1 estio orientadas a +45 em relacao a direcao x:

Calculo dos elementos da matriz de Rotacao (T) p/ camada de 45:

teta_1 = 45;
T11_45 = cosd(teta_1)^2;
T12_45 = sind(teta_1)^2;
T16_45 = 2*cosd(teta_1)*sind(teta_1);
T21_45 = sind(teta_1)^2;
T22_45 = cosd(teta_1)^2;
T26_45 = -2*cosd(teta_1)*sind(teta_1);
T61_45 = -cosd(teta_1)*sind(teta_1);
T62_45 = cosd(teta_1)*sind(teta_1);
T66_45 = (cosd(teta_1)^2) - (sind(teta_1)^2);

% Calculo dos elementos da matriz de Rotacaoao (T) p/ camada de 0:
teta_2 = 0;
T11_0 = cosd(teta_2)^2;
T12_0 = sind(teta_2)^2;
T16_0 = 2*cosd(teta_2)*sind(teta_2);
T21_0 = sind(teta_2)^2;
T22_0 = cosd(teta_2)^2;
T26_0 = -2*cosd(teta_2)*sind(teta_2);
T61_0 = -cosd(teta_2)*sind(teta_2);
T62_0 = cosd(teta_2)*sind(teta_2);
T66_0 = (cosd(teta_2)^2) - (sind(teta_2)^2);

% Calculo dos elementos da matriz de Rotacaoao (T) p/ camada de 90:
teta_3 = 90;
T11_90 = cosd(teta_3)^2;
T12_90 = sind(teta_3)^2;
T16_90 = 2*cosd(teta_3)*sind(teta_3);
T21_90 = sind(teta_3)^2;
T22_90 = cosd(teta_3)^2;
T26_90 = -2*cosd(teta_3)*sind(teta_3);
T61_90 = -cosd(teta_3)*sind(teta_3);
T62_90 = cosd(teta_3)*sind(teta_3);
T66_90 = (cosd(teta_3)^2) - (sind(teta_3)^2);


% Matriz de Rotacaoao [T] para o angulo de 45:
T_45 = [T11_45, T12_45, T16_45; T21_45, T22_45, T26_45; T61_45, T62_45, T66_45];

% Matriz de Rotacaoao [T] para o angulo de 0:
T_0 = [T11_0, T12_0, T16_0; T21_0, T22_0, T26_0; T61_0, T62_0, T66_0];

% Matriz de Rotacaoao [T] para o angulo de 90:
T_90 = [T11_90, T12_90, T16_90; T21_90, T22_90, T26_90; T61_90, T62_90, T66_90];


% Inversa da Matriz de Rotacao [T] para o angulo de 45:
T_45_i = inv(T_45);

% Inversa da Matriz de Rotacao [T] para o angulo de 0:
T_0_i = inv(T_0);

% Inversa da Matriz de Rotacaoao [T] para o angulo de 90:
T_90_i = inv(T_90);

% Transposta da Matriz de Rotacaoao [T] para o angulo de 45:
T_45_t = T_45';

% Transposta da Matriz de Rotacaoao [T] para o angulo de 0:
T_0_t = T_0';

% Transposta da Matriz de Rotacao [T] para o angulo de 90:
T_90_t = T_90';

% Inversa da Matriz Transposta da Matriz de Rotacaoao [T] de 45:
T_45_i_t = inv(T_45_t);

% Inversa da Matriz Transposta da Matriz de Rotacaoao [T] de 0:
T_0_i_t = inv(T_0_t);

% Inversa da Matriz Transposta da Matriz de Rotacaoao [T] de 90:
T_90_i_t = inv(T_90_t);

Calculo da Matriz de Rigidez de uma lamina Ortotropica angular 45 [Q_barra] - (GPa):

Q_barra_45 = T_45_i*Q*T_45_i_t;

Calculo da Matriz de Rigidez de uma lamina Ortotropica angular 0 [Q_barra] - (GPa):

Q_barra_0 = T_0_i*Q*T_0_i_t;

Calculo da Matriz de Rigidez de uma lamina Ortotropica angular 90 [Q_barra] - (GPa):

Q_barra_90 = T_90_i*Q*T_90_i_t;

Calculo da Matriz de Flexibilidade de uma lamina Ortotropica angular 45 [S_barra] - (GPa)^-1:

S_barra_45 = T_45_t*S*T_45;

Calculo da Matriz de Flexibilidade de uma lamina Ortotropica angular 0 [S_barra] - (GPa)^-1:

S_barra_0 = T_0_t*S*T_0;

Calculo da Matriz de Flexibilidade de uma lamina Ortotropica angular 90 [S_barra] - (GPa)^-1:

S_barra_90 = T_90_t*S*T_90;

% Elementos da MATRIZ DE RIGIDEZ EXTENSIONAL NO PLANO:
A1 = Q_barra_45*(z_k1 - z_k0);
A2 = Q_barra_0*(z_k2 - z_k1);
A3 = Q_barra_90*(z_k3 - z_k2);

MATRIZ DE RIGIDEZ EXTENSIONAL NO PLANO Newton/metro:

A = (A1+A2+A3)*10^9;

% Elementos da MATRIZ DE ACOPLAMENTO EXTENSAO-DEFORMACAO:
B1 = Q_barra_45*(z_k1^2 - z_k0^2);
B2 = Q_barra_0*(z_k2^2 - z_k1^2);
B3 = Q_barra_90*(z_k3^2 - z_k2^2);

MATRIZ DE ACOPLAMENTO EXTENSAO-DEFORMACAO:

B = 0.5*(B1+B2+B3)*10^9
B =
 
  -350.85  -272.11  -311.48
  -272.11   895.08  -311.48
  -311.48  -311.48  -272.11

ELEMENTOS DA MATRIZ DE RIGIDEZ A FLEXAO E TORCAO:

D1 = Q_barra_45*(z_k1^3 - z_k0^3);
D2 = Q_barra_0*(z_k2^3 - z_k1^3);
D3 = Q_barra_90*(z_k3^3 - z_k2^3);

MATRIZ DE RIGIDEZ A FLEXAO E TORCAO:

D = (1/3)*(D1+D2+D3)*10^9;

A. Calculo da MATRIZ ABBD:

ABBD = [A, B; B, D];

Calculo da matriz [abcd]

A_ast = inv(A);
B_ast = -inv(A)*B;
C_ast = -B_ast';
D_ast = D+B*B_ast;

a = A_ast+B_ast*inv(D_ast)*B_ast';
b = B_ast*inv(D_ast);
c = b';
d = inv(D_ast);

B. Calculo das deformacoes normais no plano medio (m, m):

e_zero = a*[Nx; Ny; Nxy];

Calculo das curvaturas (1/m)

K = c*[Nx; Ny; Nxy];

Calculo das deformacoeses globais no ponto medio da camada 1 (m/m):

z_m_k1 = -0.0002; % (m/m)
e_x_k1 = e_zero + z_m_k1*K;

Calculo das deformacoeses globais no ponto medio da camada 2 (m/m):

z_m_k2 = 0; % (m/m)
e_x_k2 = e_zero + z_m_k2*K;

Calculo das deformacoeses globais no ponto medio da camada 2 (m/m):

z_m_k3 = 0.0002; % (m/m)
e_x_k3 = e_zero + z_m_k3*K;

Calculo das deformacoeses locais no ponto medio da camada k = 1

e_1 = T_45_i_t*e_x_k1;

Calculo das deformacoeses locais no ponto medio da camada k = 2

e_2 = T_0_i_t*e_x_k2;

Calculo das deformacoeses locais no ponto medio da camada k = 3

e_3 = T_90_i_t*e_x_k3;

Calculo das tensoes globais no ponto medio da camada k = 1 (GPa)

T_x_1 = Q_barra_45*e_zero +Q_barra_45*z_m_k1*K;

Calculo das tensoeses globais no ponto medio da camada k = 2 (GPa)
